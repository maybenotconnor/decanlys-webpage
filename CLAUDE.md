# Decanlys Website

Single-page static website for Decanlys LLC — Connor's AI consulting brand serving Connecticut businesses.

## Tech Stack
- Pure HTML/CSS/vanilla JS (no frameworks, no build tools)
- Single `index.html` with inline styles and scripts
- Google Fonts (SUSE) as sole external dependency
- `favicon.svg` as separate file

## Design Context

### Brand Personality
**Trustworthy, Clear, Capable.** First-person voice ("I"/"my"). Tone: trusted advisor, not corporate. Speak to outcomes, not technology.

### Emotional Goal
**Relief + trust.** Visitor should feel: "Finally, someone who gets it and can actually help."

| Muted text | `#6B6B6B` |

### Typography
SUSE (Google Fonts) throughout. Bold for wordmark/hero, SemiBold for headings, Regular for body, Medium for labels. 18px base size.

### Design Principles
1. **Clarity over cleverness** — every element communicates, nothing merely decorates
2. **Spacious confidence** — generous whitespace conveys authority and calm
3. **Personal, not corporate** — human and direct, never committee-designed
4. **Substance over spectacle** — subtle animations, flat cards, clean lines
5. **Accessible by default** — WCAG AA, keyboard nav, screen readers, reduced motion

### Motion
Subtle only. Staggered hero fade-in, gentle scroll reveals, CTA hover transitions. No parallax, no particle effects, no animated backgrounds.

### Anti-patterns
Avoid: Silicon Valley startup aesthetics, corporate brochure feel, AI jargon, words like "leverage"/"synergy"/"cutting-edge"/"revolutionize"/"local".
